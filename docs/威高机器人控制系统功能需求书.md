## 威高机器人控制系统功能需求书

### 1. 任务安全与权限控制功能

**强制密码验证机制**：

* **锁定触发**：当舱门关闭并启动配送任务后，系统自动激活安全锁定状态 。
* **取物解锁**：机器人到达目的地后，选择要开启的取物舱门，输入预设的固定密码才能通过验证，允许开启舱门进行物品取用 。
* **中断保护**：在配送任务执行过程中，若需人为强行中断任务开启某一取物舱门，系统同样强制要求输入密码进行验证，防止未授权的操作干扰 。

### 2. 作业过程计量功能

**全程任务计时**：

* 系统后台自动记录配送任务的时间跨度。
* **计时逻辑**：计时起始点为“配送开始”时刻，终止点为机器人到达目的地并“再次打开舱门取用物品”的时刻 。

### 3. 核心配送模式

**标准物品配送**：

* 执行从“待机点”出发，将物品运送至指定“目的地”或“点位”的单次运输任务 。

**循环配送**：


* 机器人在预设的“循环点位”之间按照既定路线进行往复移动，执行持续的转运任务 。

**物品回收**：


* 执行从各个点位收集物品并运回“回收”点的功能 。

**慢速运行模式**：


* 针对精密或易碎物品（如内镜），系统提供特定的运动控制算法，使机器人在移动过程中保持低速和平稳，减少震动 。

### 4. 路径与点位管理功能

**自动待机与返回**：

* 任务完成后，机器人具备自动前往“返回点”或“待机点”等待下一指令的能力 。

**多点位导航**：


* 系统支持定义和识别多种类型的物理位置，包括目的地、循环点位、返回点和待机点，并规划相应路径 。

### 5. 配送模式

**显示设置**：

- 最近使用点位：开启后，若有最近使用点位则会展示在点位区域。
- 选择配送方式：
  - 一层多点：每个楼层支持选择多个目标点。
  - 多点配送：支持无楼层配送。

**时间设置**：

- 目标点停留时长：开启状态下可设置时长，关闭后则一直停留在目标点。
- 行走暂停时长。

**速度设置**：

- 配送行走速度（cm/s）。
- 返航行走速度（cm/s）。
- 慢速出发。

### 6.  舱门设置

**地面投影灯快捷控制**：

- 出发前脚踩灯光开关门：支持标准配送模式、循环配送模式使用。
- 到达后脚踩灯光自动离开：支持目标点到达场景。

### 7. 循环配送设置

**循环模式**：

- 循环次数设置。
- 运行时长设置：
  - 行走暂停时长。
  - 目标点停留时长。
  - 起点停留时长。
- 循环行走速度（cm/s）。

### 8. 远程控制

**远程呼叫**：

- 远程呼叫：开启后，空闲状态下的机器人可以被【擎朗机器人app】【擎朗呼叫器】【API接口】远程呼叫到指定点位或回收（即呼叫回收）。

**呼叫任务**：

- 呼叫循环：呼叫机器到目标点进行循环，倒计时结束后前往返回点。
- 呼叫到达：仅呼叫机器到目标点，倒计时结束后前往返回返回点。

**到达后停留时长**：

- 开启时可设置时长，关闭后则一直停留在目标点。

### 10. 声音设置

**音量控制**：

* 支持分别调节“配送背景音乐”和“语音音量”。

**配送声音**：

- 支持设置”配送背景音乐“和”取物语音播报频率“。
- 支持设置“循环配送背景音乐”和“循环配送语音播报频率”。
